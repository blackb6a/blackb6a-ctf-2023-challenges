# Challenges for Bauhinia CTF 2023

This is a repository for the challenges of Bauhinia CTF 2023.

```
└── id-challengename/
  ├── env/                    a dockerized environment that is accessible via `nc` or browsers
  │   ├── docker-compose.yml
  │   ├── chal
  │   │   ├── Dockerfile
  │   │   └── ...
  │   └── healthcheck
  │       └── main.py         unused
  ├── public/                 contains the file available for user download
  ├── src/                    whatever files that is used as auxiliary to the challenge, in whatever use.
  │                           this includes but not limited to the source code for the binary, the solution
  │                           script (or writeup) that is considered to solve the challenge
  ├── chal.yml                contains the metadata for the challenge (category, author, flag...)
  ├── README.md               the README file that contains a brief summary (title, description) of the challenge
  └── cover.jpg               the cover image that show on youtube page, should be around 16:9

```
